# FriendFindr
A system to passively track the location of individual users by collecting mac-address probe requests.
## Goal
Passively track people with publicly accessible data.

# Dependencies

##Ubuntu Server (14.04 or higher)
-MySQL
-NodeJS

##Raspberry Pi Node
-python 2.7
-python-scapy
